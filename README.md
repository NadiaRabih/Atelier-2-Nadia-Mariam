rapport d’atelier JavaScript

Lors de cet atelier de JavaScript, nous avons appris comment développer une application en utilisant uniquement le langage JavaScript, 
sans avoir recours à une base de données. L’objectif était de comprendre comment gérer les données directement dans la mémoire du navigateur à travers des tableaux et des objets.
Nous avons également étudié la transition du modèle de programmation basé sur les fonctions « constructor » vers une approche plus moderne utilisant les classes ES6. 
Cette évolution nous a permis d’organiser le code de manière plus claire, plus structurée et plus proche du paradigme orienté objet. 
Enfin, nous avons mis en place l’héritage entre les classes afin de réutiliser le comportement commun, d’éviter la duplication du code et de mieux structurer l’application. 
Cet atelier nous a aidés à maîtriser les bases de la POO en JavaScript et à comprendre comment construire une architecture propre et évolutive.
